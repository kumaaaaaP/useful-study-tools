沈殿の色

水酸化物の沈殿

  • Fe(OH)₂: (1)緑白色
  • Fe(OH)₃: (2)赤褐色
  • Cu(OH)₂: (3)青白色
  • Zn(OH)₂: (4)白色
  • Al(OH)₃: (5)白色
  • Cr(OH)₃: (6)灰緑色
  • Ni(OH)₂: (7)緑色
  • Co(OH)₂: (8)桃色
  • Mn(OH)₂: (9)淡桃色
  • Ag₂O (AgOHは不安定): (10)褐色

硫化物の沈殿

  • CuS: (11)黒色
  • PbS: (12)黒色
  • HgS: (13)黒色
  • CdS: (14)黄色
  • ZnS: (15)白色
  • FeS: (16)黒色
  • MnS: (17)淡桃色
  • NiS: (18)黒色

その他の沈殿

  • AgCl: (19)白色
  • AgBr: (20)淡黄色
  • AgI: (21)黄色
  • PbCl₂: (22)白色
  • PbI₂: (23)黄色
  • BaSO₄: (24)白色
  • CaSO₄: (25)白色
  • CaCO₃: (26)白色

錯イオンの色

銅(Cu)の錯イオン

  • [Cu(NH₃)₄]²⁺ (テトラアンミン銅(II)イオン): (27)深青色
  • [Cu(OH)₄]²⁻ (テトラヒドロキソ銅(II)イオン): (28)深青色
  • Cu(H₂O)₆²⁺ (水和イオン): (29)青色

銀(Ag)の錯イオン

  • [Ag(NH₃)₂]⁺ (ジアンミン銀(I)イオン): (30)無色
  • [Ag(S₂O₃)₂]³⁻ (ジチオ硫酸銀(I)イオン): (31)無色
  • [Ag(CN)₂]⁻ (ジシアノ銀(I)イオン): (32)無色

亜鉛(Zn)の錯イオン

  • [Zn(NH₃)₄]²⁺ (テトラアンミン亜鉛(II)イオン): (33)無色
  • [Zn(OH)₄]²⁻ (テトラヒドロキソ亜鉛(II)イオン): (34)無色

アルミニウム(Al)の錯イオン

  • [Al(OH)₄]⁻ (テトラヒドロキソアルミン(III)イオン): (35)無色

クロム(Cr)の錯イオン

  • CrO₄²⁻ (クロム酸イオン): (36)黄色
  • Cr₂O₇²⁻ (二クロム酸イオン): (37)橙色
  • Cr³⁺ (クロム(III)イオン): (38)緑色

鉄(Fe)の錯イオンと呈色反応

  • Fe²⁺ + [Fe(CN)₆]³⁻ → 濃青色沈殿 (39)ターンブルブルー
  • Fe³⁺ + [Fe(CN)₆]⁴⁻ → 濃青色沈殿 (40)ヘキサシアノ鉄(II)酸鉄(III) (紺青)
  • Fe³⁺ + SCN⁻ → (41)血赤色
  • Fe³⁺ + フェノール → (42)紫色

コバルト(Co)の錯イオン

  • Co²⁺ (水和イオン): (43)桃色
  • [Co(NH₃)₆]³⁺ (ヘキサアンミンコバルト(III)イオン): (44)黄褐色
  • CoCl₄²⁻ (テトラクロロコバルト(II)イオン): (45)青色

ニッケル(Ni)の錯イオン

  • Ni²⁺ (水和イオン): (46)緑色
  • [Ni(NH₃)₆]²⁺ (ヘキサアンミンニッケル(II)イオン): (47)青紫色
  • Ni²⁺ + ジメチルグリオキシム → (48)赤色沈殿

重要な反応まとめ

両性元素(酸・塩基両方に溶ける)

  • Zn(OH)₂ + (49)2OH⁻ → [Zn(OH)₄]²⁻
  • Al(OH)₃ + (50)OH⁻ → [Al(OH)₄]⁻
  • Cr(OH)₃ + (51)3OH⁻ → [Cr(OH)₆]³⁻

ハロゲン化銀の溶解性

  • AgCl: (52)NH₃に可溶
  • AgBr: (53)濃NH₃に可溶
  • AgI: (54)NH₃に不溶

クロム酸と二クロム酸の平衡

  • 2CrO₄²⁻ + 2H⁺ ⇄ (55)Cr₂O₇²⁻ + H₂O
  • 酸性: (56)橙色 (Cr₂O₇²⁻)
  • 塩基性: (57)黄色 (CrO₄²⁻)

過マンガン酸カリウムの反応

  • MnO₄⁻ (酸性溶液): (58)赤紫色 → Mn²⁺ (淡桃色)
  • MnO₄⁻ (中性・塩基性): (59)MnO₂ (黒褐色沈殿)
